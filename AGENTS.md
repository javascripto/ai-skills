# AGENTS

## Objetivo
Diretrizes globais para agentes neste repositório.

## Convenções Globais

Estas regras se aplicam a todas as skills e agentes:

- Stack padrão: React + TypeScript + Vite
- Gerenciador de pacotes padrão: npm
- Código em inglês
- Estrutura limpa e organizada
- Evitar prompts interativos
- Preferir comandos não-interativos
- Evitar ferramentas experimentais
- Sempre documentar decisões
- Preferir instruções não-destrutivas em projetos existentes (fazer merge em vez de sobrescrever arquivos inteiros)

## Notas
- Este arquivo é a fonte única de verdade para convenções globais.
- Atualize este documento de forma concisa sempre que uma regra global mudar.
